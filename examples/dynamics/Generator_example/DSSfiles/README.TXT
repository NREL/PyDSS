Requires version 7.6.5.35 or later  (Feb 18, 2017)

Instructions for running the Induction Machine test for the new INDMACH012 model.

1. Open Run.DSS in OpenDSS
2. Execute the script

It will 

1. Compile the Master.DSS file 
2. Execute a power flow  (You can review the solution after this step).
3. Execute 5000 steps of a dynamic simulation with a SLG fault applied.
